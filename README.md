# Progreso Personal

Aplicación web desarrollada con Laravel que permite a los usuarios registrar y visualizar su progreso personal en distintos ámbitos como hábitos, actividad física, objetivos y lecturas.

## Tecnologías utilizadas

- PHP
- Laravel
- MySQL
- Blade
- Bootstrap

## Funcionalidades

- Registro y autenticación de usuarios
- Gestión de hábitos
- Registro de progreso
- Visualización de estadísticas
